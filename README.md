# استفتاء إعادة الانتخابات - مشروع تفاعلي ساخر

## الفكرة
موقع ويب تفاعلي ساخر حول "إعادة الانتخابات":  
- زر "نعم" يهرب ولا يمكن الضغط عليه.  
- زر "لا" عند الضغط عليه يظهر شريط نتائج يمتلئ تدريجياً.  
- بعد انتهاء العد تظهر رسالة **"نشوفك في 2069"**.
